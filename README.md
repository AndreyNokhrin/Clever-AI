# Clever AI

Enterprise SaaS platform deploying AI voice, SMS, and email agents 
to automate off-market land acquisition for enterprise homebuilders 
and energy infrastructure developers.

**Anchor customer:** Lennar — #1 US homebuilder  
**Pilot results:** $40K spend → $122.5M pipeline value → 3x ROI  
**Traction:** $500K ARR in 90 days  

## What this repo contains
Prompt engineering infrastructure for Clever AI's landowner 
outreach agents — the foundation of our AI conversation layer 
trained on 100,000+ real landowner calls.

## Stack
- Claude (Anthropic) — core reasoning layer
- Twilio — voice/SMS/email delivery  
- Make + N8N — workflow automation
- Python — agent orchestration
- Claude Code — primary development tool
